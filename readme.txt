Description:
This app uses your microphone to send a message to ChatGPT. The response will then either be read out with either Eleven Labs or text to speech.

Setup:
1) Pip install -r requirements.txt
2) Set your OPENAI_API_KEY as an environment variable
3) Optionally set your ELEVEN_LABS_KEY as an environment variable

running:
When running app.py, alt is the default key to turn on your microphone
